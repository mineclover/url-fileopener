# Getting Started

> 🔗 **문서 위치**: [INDEX.md](../INDEX.md) > Guides > Getting Started

Effect CLI를 처음 시작하는 분들을 위한 단계별 가이드입니다.

## 🚀 빠른 시작

### 1. 설치 및 실행
```bash
# 의존성 설치
pnpm install

# 개발 모드 실행
pnpm dev --help
```

### 2. 첫 번째 명령어 실행
```bash
# 현재 디렉토리 목록
pnpm dev ls

# 파일 내용 읽기
pnpm dev cat package.json

# 파일 검색
pnpm dev find . "*.ts"
```
